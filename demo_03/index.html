这是一个 Webpack 的入门教程

这个 DEMO 主要讲述 webpack 中 module 和 loader 的使用

其实对于 module 参数，我们用的最多的就是 rules，它用来匹配文件并对其做相应的处理

rules 中的 test 支持正则，这就能帮组我们完全很多工作

而 use 则调用 loader 来实现文件处理

上一节中我们通过 webpack 实现了自动打包，但是打包完成的 js 文件并不完全是 ES5 标准的，所以在不同浏览器上的表现会很不一样

而这次，我们通过 module 调用 loader 来自动完成 ES6 的转换，引用的插件主要是：

  • babel
  • babel-core
  • babel-loader
  • babel-preset-es2015
